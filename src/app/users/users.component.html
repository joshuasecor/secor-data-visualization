A more useful report on Users

A method on the service...

appService.getStreamsPerUserByCountry();

...could make a call to an API endpoint...

https://api.umg.com/get/streamsPerUserByCountry

...that queries the data as needed...

SELECT country, (SUM(streams) / SUM(users)) as avg_streams
FROM user_streams
GROUP BY country
ORDER BY avg_streams DESC;

...to return the average streams per user for each country.
