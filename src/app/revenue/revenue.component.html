A more useful report on Revenue

A method on the service...

appService.getRevenuePerUserByConsumerGroup();

...could make a call to an API endpoint...

https://api.umg.com/get/revenuePerUserByConsumerGroup

...that queries the data as needed...

  SELECT consumergroup, (SUM(revenue) / SUM(users)) as avg_revenue
  FROM user_revenue
  GROUP BY consumergroup
  ORDER BY avg_revenue DESC;

...to return the average revenue per user for each consumer group (premium vs. ad-funded).
